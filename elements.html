Gomoku專案

這是一個使用 C# 與 Windows Forms (WinForms) 製作的桌面應用程式，模擬傳統五子棋對戰。專案主要目標是透過物件導向的方式實作雙人互動遊戲，並練習圖形介面設計與基礎邏輯判斷。

使用技術

程式語言：C#
開發平台：.NET Framework（Windows Forms）
設計模式：物件導向設計（OOP）
介面控制元件：PictureBox、Panel 等

已完成的功能

使用 WinForms 製作視窗化遊戲介面
支援雙人對戰，黑白棋子輪流落子
使用 PictureBox 顯示棋子圖案（動態加入控制元件）
自訂棋子類別（BlackPiece、WhitePiece）提高可維護性
自訂 Game 類別管理棋盤、座標、勝利邏輯判斷
落子後立即檢查是否有五子連線
將座標轉換為陣列索引與畫面位置對應
Form1 作為主視窗，管理 UI、事件處理與控制流程

開發重點與挑戰

學會將程式邏輯抽象化成類別與方法，提高程式可讀性與模組化程度 熟悉 Windows Forms 的事件機制與控制項操作（如：滑鼠事件、動態加入控制元件） 練習二維陣列與迴圈結合的判斷邏輯 強化除錯技巧（處理坐標對應、勝利判定邏輯錯誤等）

未來計畫

加入遊戲設定選項（如棋盤大小、棋子顏色等）
優化介面設計與使用者體驗
撰寫完整的使用說明文件與程式註解
